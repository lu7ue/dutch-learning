## 2. 元音

荷兰语有5个元音字母： a, e, i, o, u

### 2.1 长元音（写成两个字母）

aa → 啊——  
ee → 诶——  
oo → 哦——  
uu → 于——  
ie → 衣——

## 3. 开音节与闭音节

开音节（open syllable）是指音节以元音结尾，即 CV（辅音+元音）或单元音，如 ma，ro-de，Eu-ro-pa。开音节里的单个元音通常读长音。  
闭音节（closed syllable）是指音节以辅音结尾，即 CVC，如 man，ram，zon。闭音节里的单个元音通常读短音。

> 闭音节 + 单元音 = 短音  
> 闭音节 + 双元音 = 长音  
> 开音节 + 单元音 = 长音

## 4. 拼写规则

规则1：闭音节 + 长音 → 双写元音，如 maan，rood，buur  
规则2：闭音节 + 短音 → 单写元音，如 man，bon，wit，zit  
规则3：开音节 + 长音 → 单写元音，如 ma-nen，ro-de，bu-ren  
规则4：短音变复数或变形时，必须保持闭音节 → 双写辅音，如 man → man-nen，bon → bon-nen，wit → wit-te，zit → zit-ten

## 5. 音节划分

单个中间辅音 → 在辅音前分开，如 ma-nen，ra-men，zo-nen  
两个中间辅音 → 在中间分开，如 man-nen，bus-sen，ken-nen

## 6. 辅音变化规则

在变形时（通常词尾加 -en），先看是否满足下面三个条件：

1）有长元音 aa / ee / oo / uu（不包括 ie）  
2）长元音后只有一个 f 或 s  
3）后面加的是 -en（e 是元音）

如果满足，则：  
1）长元音 + 单 f → f 变 v ，即 graaf → graven  
2）长元音 + 单 s → s 变 z ，即kaas → kazen

如果不满足，则：  
1）f / s 不变  
2）若是短元音 + 单辅音，为保持短音，需要双写辅音 + en ，即 stof → stoffen，kus → kussen

注意：单词结尾不能双写辅音（错误：kenn, mann；正确：ken, man）
